======================================================================
  Konica Minolta Universal Printer Driver v2.3.1
  Installation Guide
======================================================================

Thank you for choosing Konica Minolta.

This package contains the Universal Printer Driver for all
supported Konica Minolta bizhub and magicolor series printers.

INSTALLATION INSTRUCTIONS:

1. Close all open applications
2. Double-click setup.exe
3. Follow the on-screen installation wizard
4. Select your printer model from the list when prompted
5. Complete the installation and restart if requested

SILENT INSTALLATION (for IT administrators):

  setup.exe /S /v"/qn"

SYSTEM REQUIREMENTS:

- Windows 10 (32/64-bit), Windows 11
- Windows Server 2016/2019/2022
- 150 MB free disk space
- 512 MB RAM (2 GB recommended)
- Administrator privileges

UNINSTALL:

- Control Panel > Programs > Konica Minolta Universal Driver > Uninstall
- Or run: uninstall.exe
